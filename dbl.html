IMPORTANT

Run s/setup or @ServerStats setup to start the bot and s/counter or @ServerStats counter to see all available counters.

COMMANDS

s/help [command]

Send all the commands if you don't put a command after it else it will if info about the command.

s/setup [channel type]

This command will setup the bot and create all the default counters that the bot has.

s/counter [action] [counter] [channel type] [additional]

With this command you can turn manage all counters.

s/welcome <action> [additional]

This will create a join and leave log channel.

s/goal <action> [type] [channel type] [announce channel]

This will create a goal counter and message.

s/members [members/bots/role-ID/@role-mention]

This command will send a file with all the members in your server.

s/debug [<ID>/server/gain]

This command will send you debug info, this can help fix problems faster.

s/reset

With this command you can reset the counters.

s/check

This will check if the bot has the right permissions in the server and counter channels.

s/category [action]

This will create a new category and puts the counter under it. (you don't need a category)

s/serverinfo

Sends info about the server.

s/userinfo [ID/@mention]

Sends info about an user.

s/roleinfo <ID/@mention>

Sends info about an role.

s/channelinfo [ID/#mention]

Sends info about an channels.

s/inviteinfo <URL/code>

Sends info about an invite.

s/avatar [ID/@mention]

Sends the avatar form an user.

s/emojiinfo <ID/emoji/emoji name>

This command will send information from the emoji that you have provided.

s/prefix <new prefix/reset>

This command will change the prefix for your server.

s/command <enable/disable> <command>

This command will disable / enable a command, currently you can only disable "info" commands.

s/bot

Sends info about the bot.

s/data

This will send all the data saved about your server.

s/ping

Sends a pong message with latency.

s/invite

Sends an invite link for the bot.

s/vote

Sends instructions on how to vote for the bot.

s/donate

Sends instructions on how to donate.

s/support

Sends info about how to get help.

s/partners

Sends info about our partners.

s/customize

Sends instruction of how to change the text of the counter.

Normal Commands

s/help

Sends a DM message with all the available commands.

s/customize

Sends a message about how to change the counters.

s/ping

Sends a pong message in the channel with latency in MS.

s/invite

Sends an DM message with a invite link for the bot.

s/vote

Sends a DM message with a link to the vote page and instructions on how to vote.

s/donate

Sends a DM message with a link to donate and instructions on how to do it.

s/error

Sends a DM message with info about what-to-do if there is a problem.

Info Commands

s/info

Sends a message with infomation about the bot like: version, ram, shards info and how many server he is in.

s/serverinfo

Sends information about the server you use the command in.

s/userinfo <ID/mention>

Sends information about the user you have mentioned or the ID you have given.

s/avatar <ID/mention>

Sends the avatar from the user you have mentioned or the ID you have given.

Admin Commands

s/setup

A command to set up the bot (Can only be used ones), this will create some channels.

s/refresh

Force refresh the counters (only use this if you thinks that the data is wrong).

s/data

Shows you the data from FireStore (only shows your server data).

s/remake

This will delete the old channel (if they're still there) and replace them with new ones so the channel will be set to default.

s/category <ID>

Change or set the category where the channels will be created for the remake command.

s/channel <all/members/bots> <ID/mention>

Change one of the counters to another channel (can be a text or voice channel), keep in mind that there must be a number in the name to change.

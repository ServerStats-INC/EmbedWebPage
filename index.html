Bot Features

Fancy member counter

This bot will show all your members and bot in a fancy way, one's set up the bot will automatically update the counters.

Easy to setup

Some bots are a pain to set up but not this bot but, the bot will automatically run de setup if a member joins or leaves or you can run the setup by using s/setup.

Custom text

You don't like the default text? No problem, just changes the channel name how you normally would.
IMPORTANT Place a number where you want the counter to be, like this: My 1 bro's (the 1 will automatically change).

24/7 Uptime and lite

The bot is build to run 24/7 because is very lite weight and it runs on a strong VPS, the only downtime will be for bot and server update (normally not longer than 10 min).

Firestore database

The bot uses Google FireStore to save all the data, you can request the data by typing s/data.

And more...
